## Project Overview
MSU Surplus Tracker is a web-based system designed to help Midwestern State University track surplus assets and manage disposal processes.

## Tech Stack
Frontend: Next.js
Backend: FastAPI
Database: PostgreSQL

## Key Features
- Surplus item submission portal
- Barcode scanning for warehouse intake
- Asset lifecycle tracking
- AI-assisted disposal recommendations
- Audit trail logging
