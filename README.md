# MSU Surplus Tracker

MSU Surplus Tracker is a web-based system designed to help Midwestern State University track surplus assets and manage disposal processes.

Currently at Midwestern State University, the process of disposing and recycling university hardware, equipment, furniture and other materials is largely manual. This project introduces MSU Surplus Tracker, a centralized surplus tracking system that streamlines the process by providing a single platform for logging, managing, and documenting surplus items. With this system in place, bulky paperwork can be reduced to simple scanning and standardized digital records, minimizing inefficient tracking and data inaccuracies.

## Tech Stack
- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL

## Key Features
- Surplus item submission portal
- Barcode scanning for warehouse intake
- Asset lifecycle tracking
- AI-assisted description/analysis support
- Audit trail logging
